# Anu PlayWrite Automation

This repository contains JavaScript learning examples and notes organized by chapter. It appears to be focused on Playwright automation basics and JavaScript fundamentals.

## Repository Structure

- `chapter_01_Basics/`
  - Basic JavaScript examples and Playwright setup scripts.
  - Files include foundational exercises such as `01_Basics.js` through `05_JS_Basics.js`.

- `chapter_02_Javascript_Concepts/`
  - JavaScript concept scripts for deeper learning and verification.
  - Includes `05_JS_Basics.js` and related example files.

- `chapter_03_Identifier_Literals/`
  - Identifier and literal rules with examples.
  - Contains scripts like `06_Identifier_Rules.js`, `07_Identifier_Part2.js`, `08_Comments.js`, `js_identifier_rules.js`, and a VS Code shortcut reference file.

## Recommended Usage

1. Open this repository in VS Code or your preferred editor.
2. Review the JavaScript files in the chapter folders.
3. Run the example scripts using Node.js if they are standard JavaScript code.

## Notes

- `VS_Code_keyboard_shortcut_windows.md` contains useful VS Code keyboard shortcuts for Windows.
- The repository does not currently include a package manager manifest, so install dependencies manually if you add any automation or Playwright packages.
